Requests

Estrutura de url


          esquema://domínio/caminho/recurso?query_string

Exemplo


          import requests

          url = 'https://pixabay.com/pt/photos/'
          payload = {'q': 'gatinhos felizes'}

          r = requests.get(url, params=payload)
          print(r.status_code)

BeautifulSoup

O que é HTML?

  • Abreviação para Hyper Text Markup Language.
  • Serve para estruturar as páginas da internet.
